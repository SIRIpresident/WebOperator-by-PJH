> CSS

color 속성 사용법

 color 속성은 글씨의 색을 바꿀 때 사용된다.
 color: red;같은 형식으로 사용된다.

font-size 속성 사용법

 font-size 속성은 글씨의 크기를 바꿀 때 사용된다.
 font-size: 30px;같은 형식으로 사용된다.

font-family 속성 사용법

 font-family 속성은 글씨체를 바꿀 때 사용된다.
 font-family: 'Song Myung', serif;같은 형식으로 사용된다.

text-align 속성 사용법

 text-align 속성은 글씨의 수평 정렬을 설정할 때 사용된다.
 center, left, right의 속성을 지정할 수 있다.
 text-align: center;같은 형식으로 사용된다.

background-color 속성 사용법

 background-color 속성은 배경색을 변경할 때 사용된다.
 background-color: blue;같은 형식으로 사용된다.

margin 속성 사용법

 margin 속성은 바깥쪽의 여백을 설정할 때 사용된다.
 특정 수치를 대입하거나 auto로 지정한다.  margin-top, margin-bottom, margin-left, margin-right처럼
 한쪽만 적용되게 할 수도 있다.
 margin: 5px;같은 형식으로 사용된다.

padding 속성 사용법

 padding 속성은 안쪽의 여유공간을 설정할 때 사용된다.
 특정 수치를 대입하여 사용한다.
 padding-top, padding-bottom, padding-left,
 padding-right 처럼 한쪽만 적용되게 할 수도 있다.
 padding: 5px;같은 형식으로 사용된다.

width 속성 사용법

 width 속성은 너비를 지정할 때 사용된다.
 px나 %를 단위로 값을 지정해줘야한다.
 width: 500px;같은 형식으로 사용된다.

height 속성 사용법

 height 속성은 높이를 지정할 때 사용된다.
 px나 %를 단위로 값을 지정해줘야 한다.
 height: 600px;같은 형식으로 사용된다.

border 속성 사용법

 border 속성은 외각선을 지정할 때 사용된다.
 색과 너비, 외각선의 형태를 지정해줘야한다.
 border-top, border-bottom, border-left, border-right
 처럼 한쪽만 적용시킬 수 있다.
 border: 3px dotted black;같은 형식으로 사용된다.

border-radius 속성 사용법

 border를 지정해 준 뒤 모서리를 둥글게 만드는 속성이다.
 px나 %를 단위로 값을 지정해준다.
 border-radius: 10px같은 형식으로 사용된다.

display 속성 사용법

 display속성은 태그의 특성을 바꿔주는 속성이다.
 none, block, inline, inline-block 등의 값을
 지정해 줘야 한다.
 - none : 태그를 사라지게 한다.
 - block : div처럼 자동 줄바꿈과 크기지정을 가능하게 한다.
 - inline : span처럼 자동 줄바꿈과 크기지정이 불가능하다.
 - inline-block : 크기지정은 가능하고 줄바꿈은 불가능하다.
 display: inline-block같은 형식으로 사용된다.

float 속성 사용법

 float 속성은 이미지를 웹페이지의 어느곳에 띄울것인디
 정하는 속성이다.
 inherit, left, right, none 등의 값을
 지정해줘야 한다.
 - inherit : 부모 요소에서 상속한다.
 - left : 왼쪽에 부유하는 블록박스를 생성한다.
 - right : 오른쪽에 부유하는 블록박스를 생성한다.
 - none : 요소를 부유시키지 않는다.
 float: left;같은 형식으로 사용된다.

clear 속성 사용법

 clear 속성은 float속성을 삭제하는 속성이다.
 clear: left;는 float: left;를 취소하고,
 clear: right;는 float: right;를 취소하고,
 clear: both;는 둘 다 취소한다.

position 속성 사용법

 position 속성은 레이아웃을 배치하거나 객체를 위치시킬 때
 사용할 수 있는 속성이다.
 - static : 정적이게 만드는 속성
 - relative : 위, 아래, 좌, 우를 위치를 설정 가능
 - absolute : 어떤 위치나 상관없이 위치시킬 수 있다.
 - fixed : 문서를 스크롤해도 처음 지정한 화면의 위치에
           남아있다.
 - sticky : 스크롤을 내리다가 지정된 오프셋에 도달하면
            고정된다.
position: fixed;같은 형식으로 사용된다.

z-index 속성 사용법

 z-index 속성은 레이아웃의 배치 순서를 지정하는 속성이다.
 position속성이 relative, absolute, fixed로 적용된 요소에서만
 작동한다.
 뒤에 숫자를 지정해주면 숫자가 가장 큰 레이아웃이 앞으로 나온다.
 z-index: 3;같은 형식으로 사용된다.

opacity 속성 사용법

 opacity 속성은 요소의 불투명도를 지정하는 속성이다.
 뒤에 0.0에서 1.0까지의 수를 지정해준다.
 뒤에 지정된 숫자가 작을수록 더 투명해진다.
 opacity: 0.7;같은 형식으로 사용된다.

> JavaScript

주석처리하는 방법

 주석처리한 부분은 실제 코드 실행시 무시된다.
 자바스크립트에서 주석을 붙이는 방법은
 코드 주석처리를 시작하는 부분에 //를 붙이거나(한 줄 주석)
 /*와 */ 사이에 주석처리할 내용을 넣으면 된다.
 //이곳은 주석입니다.또는 /*이곳은 주석입니다*/같은 형식으로
 사용된다.

변수 선언하는 방법

 자바스크립트의 변수는 변수의 형식에 상관없이 var로 통일한다.
 var num = 0;이나 var string = 'Hello World!';
 과 같은 형태로 사용한다.

alert(''); 함수 사용법

 alert('');함수는 웹페이지에서 알림창을 띄우는 함수이다.
 alert('');안에 알릴 메세지를 입력하면 된다.
 alert('Hello World');같은 형식으로 사용한다.

prompt(''); 함수 사용법

 prompt('');함수는 창을 띄워 값을 입력받는 함수이다.
 var inputText = prompt('당신의 이름을 입력하세요.');같은
 형식으로 사용한다.

confirm(''); 함수 사용법

 confirm('');함수는 메세지를 출력하고 확인/취소버튼을 가진
 다이얼로그를 출력한다. '확인'버튼을 누르면 true,
 '취소'버튼이나 강제로 다이얼로그를 닫으면 false를 리턴한다.
 var choice = confirm('이해하셨습니까?');같은 형태로
 사용한다.

연산자의 종류

 JavaScript에는 여러 연산자가 있다.
 - 산술연산자 : +(덧셈) -(뺄셈) *(곱셈) /(나눗셈) %(나머지)
 - 증감연산자 : ++(1 더함) --(1 뺌)
 - 비트연산자 : & | ^ ~
 - 대입연산자 : = *= += -= !=
 - 비교연산자 : > < >= <= == !=
 - 논리연산자 : && ||
 var num = 5 + 7같은 형식으로 사용한다.

조건문 만들기

 조건문의 기본 형식은 if (비교연산식) { 실행할 코드 }이다.
 추가로 else if (비교연산식) { 실행할 코드 }, else { 실행할 코드 }  을 붙인다. else if문은 if문의 비교연산식이 거짓일 때 실행되고,
 else문은 if문과 모든 else if문이 거짓일 때 실행된다.

웹사이트 연결하기

 location.href = url;같은 형식으로 사용하고,
 url 부분에 웹사이트의 링크를 입력한다.
 location.href = 'http://google.com';같은 형식으로 사용한다.

while문 사용법

 while문은 while(비교연산식) { 실행할 코드 }의 형식으로 사용한다.
 실행할 코드는 비교연산식이 참일경우 무한반복되고, 거짓일 경우 멈춘다.
 while(true) { alert('Hello World!!!'); }같은
 형식으로 사용한다.

for문 사용법

 for문은
 for (변수의 초기값; 비교연산식; 실행할 코드1) { 실행할 코드2 }
 같은 형식으로 사용한다. 예를 들어서
 for (var i = 0; i < 5; i++) { alert('Hi'); }는
 i의 초기값을 i로 설정하고 i가 5보다 작을동안
 i를 1씩 더한다는 것이므로 0부터 4까지, 즉 실행할 코드2가
 5번 반복된다.

객체 사용법

 객체는 var userObj = { 키1 : 값1, 키2 : 값2 };의 형식으로
 사용한다. 여기에서 키와 값을 합쳐서 속성이라 부르는데
 각 속성은 쉼표로 구분한다. 값에는 다양한 자료형과 함수 등이
 들어갈 수 있다.
 var userData = {
     age : 17,
     sayHi : function() {
         alert('Hi');
     }
 }
 처럼 선언하고, 객체 안의 값은 alert(userData.age);나
 userData.sayHi();처럼 호출한다.

document 객체 사용법

 document 객체에는 매우 다양한 프로퍼티가 있다.
 주요 프로퍼티로와 메소드는
 <프로퍼티>
 - location : 문서의 URL정보를 가진 location객체의 레퍼런스
 - domain : 서버의 도메인 이름
 - lastModified : "MM/DD/YYYY hhmmss"형식의 문서의
                  최근 수정시간
 - readyState : 문서의 현재 로딩상태를 나타내는 문자열
 - title : 문서의 탭 제목. 없으면 빈 문자열을 출력
 - body : body객체에 대한 레퍼런스
 - head : head객체에 대한 레퍼런스
 - URL : 현제 문서의 URL
 <메소드>
 - getElementsByTagName() : ()안의 태그이름을 가진 모든 태그
                            의 컬렉션 리턴
 - getElementById() : ()안의 class속성값을 가진
                              모든 태그의 컬랙션 리턴
 - getElementByName() : ()안의 name속성 값을 가진 모든 태그
                       의 컬랙션 리턴
 - querySelector() : 서낵자로 가장 먼저 선택되는 DOM객체 리턴
 - querySelectorAll() : 선택자로 DOM객체 컬랙션 리턴
 프로퍼티는 alert('document.location');같은 형식으로 사용하고
 메소드는 document.getElementById('selected').style.backgroundColor = 'red'  같은 형식으로 사용한다.
